# **capacitacao_de_python**

Este é um projeto de uma multicalculadora O programa permite que o usuário faça cálculos envolvendo as quatro operações matemáticas; calcule o troco a ser recebido ao realizar uma compra; calcule a resistência obtida ao usar vários resistores juntos, seja em sértie ou paralelo.

============================================================================================

**Estruturação do código**

• calculadora_basica.py (Responsável pela calculadora de expressões matemáticas)

• calculadora_resistencia.py (Responsável pelo cálculo da resistencia)

• calculadora_troco.py (Responsável pelo cálculo de troco)

• main.py (responsável pelo menu e pelo loop principal)

============================================================================================
**funcionameno**

• Menu:
O usuário consegue acessar as opções no menu digitando o número relacionado à opção.

• (1) calculadora de troco:
Para utilizar a calculadora de troco o usuário deve informar somente o valor da compra e o valor pago e o programa irá dizer qual o valor do troco e as notas e moedas necessárias para passar o troco da forma mais eficiente.

• (2) calculadora básica:
Para utilizar a calculadora básica o usuário deve informar um número, uma operação (adição, subtração, multiplicação ou divisão) e o outro número. Nessa função o usuário tem a opção de voltar ao menu digitando "S".

• (3) calculadora de resistência:
Para utilizar a calculadora de resistência o usuário deve informar o tipo da montagem que será feita (série ou paralelo), quantos resistores serão utilizados na montagem e o valor da resistência de cada um deles.

• (0) sair:
Ao escolher essa opção o programa exibe uma mensagem de encerramento e fecha.
